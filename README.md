# HDFC-M&A-Analysis
M&amp;A analysis of HDFC Bank and HDFC Ltd merger including strategic rationale, synergies, financial impact and risk assessment. Demonstrates understanding of mergers and acquisitions and deal evaluation techniques used in investment banking.

🤝 M&A Analysis — HDFC Bank & HDFC Ltd Merger

Author: Yash Bagul

This project analyzes the merger between HDFC Bank and HDFC Ltd, one of the largest financial sector mergers in India.
The analysis focuses on strategic rationale, synergies and financial impact of the transaction.

🔍 Project Scope

- Understanding the merger structure

- Strategic rationale behind the deal

- Revenue and cost synergies

- Financial impact and EPS effect

- Risks and long-term benefits

🛠 Tools Used

- Financial analysis

- Strategic analysis

- PowerPoint presentation

📁 Files Included

- Detailed M&A analysis document (PDF)

- Presentation slides

🎯 Skills Demonstrated

- M&A analysis
- Strategic finance
- Corporate finance
- Deal evaluation
